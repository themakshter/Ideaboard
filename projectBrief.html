User Stories

  • As a student I want to keep track of tasks for each of my modules, so that I do not lose track of my work
  • As a user I want to be able to make notes in rich text such as headings and bullet points
  • As a user I want to create a board to store and organise notes, so that I can get an overview with just one glance
  • As a user I want to have access to my boards from anywhere, so that I can always check my notes
  • As a user I want an unobtrusive user interface, so that I can view my notes without becoming distracted
