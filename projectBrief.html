COMP6205 Project Brief

Idea Board

Introduction

Many people use boards to stick their notes on. This helps them visualise their train of thought and also helps them keep their ideas organised. An example of such a board is shown below:


We will attempt to create a note-taking application which will model after such boards. Users will be able to add idea boards and then add notes on them. These notes will be divided into different columns which can be named by users. The notes will also have a reasonable degree of customisation such as basic text formatting and custom note colours.

User Stories

  1. As a student, I want to keep track of tasks for each of my modules, so that I do not lose track of my work.
  2. As a user, I want to be able to make notes in rich text such as headings and bullet points, so that I can customise my notes as I like .
  3. As a user I want to create a board to store and organise notes, so that I can get an overview with just one glance.
  4. As a user I want to have access to my boards from anywhere, so that I can always check my notes.
  5. As a non-administrative user, I want to view and modify my idea boards but not the boards of others.
  6. As a user I want an unobtrusive user interface, so that I can view my notes without becoming distracted. (optional)
  7. As a developer, I want to make sure that the application works properly on the latest versions of all browsers, so that users can use their preferred browsers.
